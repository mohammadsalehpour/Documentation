# راهنمای نوشتن مستندات

برای نوشتن یا ویرایش مستندات در این مخزن، این اصول را رعایت کنید:

- ساده و شفاف بنویسید.
- مثال واقعی از استفاده در تیم اضافه کنید.
- در صورت امکان، دستورات را به‌صورت مرحله‌به‌مرحله (step by step) بنویسید.
- در انتهای هر صفحه، در صورت نیاز، بخش **منابع بیشتر** اضافه کنید.
- اگر صفحه مربوط به یک ابزار/تکنولوژی است، از تمپلیت «تکنولوژی» استفاده کنید.

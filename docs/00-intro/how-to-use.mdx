# چطور از این مستندات استفاده کنیم

- اگر تازه وارد تیم شده‌اید، از بخش **Onboarding** شروع کنید.
- اگر می‌خواهید یک ابزار خاص (مثل Git یا Docker) را یاد بگیرید، به بخش مربوط به آن ابزار بروید.
- اگر در حال حل یک مشکل هستید، اول بخش **Runbookها و Troubleshooting** را ببینید.
- اگر می‌خواهید بدانید ما در تیم چطور کار می‌کنیم، به بخش **فرآیندها و گردش کار** و **معماری‌ها** سر بزنید.
